## Contexto del Proyecto
--------------------------------------------------------------------------------------------------------------------------------
Este proyecto es una aplicación web de una sola página (SPA - Single Page Application) desarrollada con React para la gestión de votaciones de representantes de cursos. Permite administrar candidatos organizados por espacios académicos (FullStack, Data Science, Ciberseguridad, etc.), registrar votos de forma visual, y visualizar dashboards con estadísticas en tiempo real, como el total de votos, el candidato líder y el top 3. Además, incluye funcionalidades avanzadas de exportación de datos a CSV para su análisis en hojas de cálculo, así como la opción de realizar backups completos en formato JSON y restaurar la información desde estos archivos, garantizando la seguridad y portabilidad de los datos del proceso electoral, todo ello sin necesidad de recargar la página gracias a su arquitectura SPA.
